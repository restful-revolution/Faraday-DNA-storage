#### Project Description and Purpose
The core idea is to investigate whether **enhanced Faraday rotation**—a magneto-optical effect where the polarization of light rotates in the presence of a magnetic field—can enable reading and writing data directly in synthetic DNA films. This approach aims to bypass traditional DNA storage bottlenecks like chemical sequencing or fluorescence-based detection. Instead, it relies solely on **light and magnets** for data manipulation and retrieval.

In essence, the project explores DNA not just as a molecular archive (as in conventional DNA data storage), but as a dynamic, optically tunable medium. By leveraging Faraday effects within a potentially Faraday-caged setup (implied by the name, though not deeply detailed), it seeks a simpler, faster alternative to enzymatic or synthesis-heavy methods. The goal is photonic storage: using polarized light to encode/decode bits without invasive biology.
